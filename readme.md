# universal_informatics
 
**A Bio‑informatics “super agent” developed by Universal Mind TV, PBC (Delaware Public Benefit Corporation).**
 
This project combines Python notebooks and helper modules to analyse genomic data and mental‑health markers, producing adaptive insights. It underpins the Universal Mind TV platform (patent pending).
 
---
 
## 🚀 Features
 
- Modular Python pipeline (notebooks + `.py` helpers)
- Focused on research in **mental health, genomics, and happiness**
- Source‑available under a customised PolyForm licence
- Mobile‑friendly workflow (Juno + Working Copy on iOS)
 
---
 
## 🏁 Getting started
 
Clone the repo and replicate the exact Python environment:
 
```bash
git clone https://github.com/universalmindpbc/universalinformatics.git
cd universalinformatics
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
 
Open the notebook `universal_informatics.ipynb` to explore the workflow.
